FAQ

How do I use this?

Simply include this script and jQuery in your website as with any other external javascript scirpt and load the glitch() function. The add the class="glitch" tag to any object you want to have a glitch effect. Still stuck? Just view the code for this page.

How can I change the settings of this script?

Just define them with an overriding script. A better way of doing this is in progress.

Can I use this in *whatever*?

Yep, I'd like you to link back to me or this page if you could, but if you really don't want to then it's fine.

Does this use jQuery?

Yes. As of version 3.0 glitch.js now uses jQuery. If you wish to use glitch.js without jQuery then use a version prior to 3.0.

Why is this HTML5?

This uses the box-shadow and text-shadow CSS rule, only used in CSS 3 (HTML5).
